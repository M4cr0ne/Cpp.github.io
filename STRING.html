1. Idea chiave

Una string è una sequenza di caratteri. In pratica, è un modo ordinato e semplice per rappresentare un testo all'interno di un programma.

Con una string possiamo salvare un nome, un cognome, una città, una frase, una password, un codice e molto altro. Ogni carattere occupa una posizione e può essere letto anche singolarmente.

In pratica: una string è come una fila di lettere e simboli, ma gestita dal linguaggio come un unico oggetto.

2. Modello mentale

Anche la string, come array e vector, può essere vista come una sequenza ordinata. La differenza è che qui gli elementi sono caratteri.

4. Esempi guidati

Esempio 1 • Stampare una string

#include <iostream>
#include <string>
using namespace std;

int main(){
    string nome = "Marco";
    cout << nome << endl;
}

Qui la variabile nome contiene un testo e viene stampata normalmente con cout.

Esempio 2 • Accesso ai caratteri

#include <iostream>
#include <string>
using namespace std;

int main(){
    string parola = "Ciao";

    cout << parola[0] << endl;
    cout << parola[2] << endl;
}

La string può essere letta per indice: il primo carattere è all'indice 0.

Esempio 3 • Concatenazione

#include <iostream>
#include <string>
using namespace std;

int main(){
    string nome = "Anna";
    string cognome = "Rossi";
    string completo = nome + " " + cognome;

    cout << completo << endl;
}

Con l'operatore + possiamo unire più string in una sola.

Esempio 4 • Lunghezza del testo

#include <iostream>
#include <string>
using namespace std;

int main(){
    string password = "abc123";
    cout << password.length() << endl;
}

Il metodo length() restituisce quanti caratteri ci sono nella string.

Esempio 5 • Caso realistico: nome completo e saluto

#include <iostream>
#include <string>
using namespace std;

int main(){
    string nome, cognome;

    cout << "Inserisci il nome: ";
    cin >> nome;

    cout << "Inserisci il cognome: ";
    cin >> cognome;

    cout << "Ciao " << nome << " " << cognome << endl;
}
Questo esempio è corretto per testi senza spazi interni. Se invece vuoi leggere una frase intera, devi usare getline().

7. Errori tipici degli studenti

Errore 1 • Confondere char e string

'A' è un carattere singolo. "A" è una string.

Errore 2 • Usare cin per leggere una frase intera

cin si ferma allo spazio. Per leggere una frase completa bisogna usare getline().

Errore 3 • Pensare che length() restituisca l'ultimo indice

Se length() vale 5, gli indici vanno da 0 a 4.

Errore 4 • Uscire dai limiti della string

string testo = "ciao";
cout << testo[4]; // sbagliato: ultimo indice valido è 3
Gli indici validi vanno sempre da 0 a length()-1.

Errore 5 • Dimenticare #include <string>

In molti ambienti funziona lo stesso, ma è corretto includere sempre la libreria in modo esplicito.

10. Riepilogo finale

  • La string serve a memorizzare testi.
  • È una sequenza ordinata di caratteri.
  • Puoi leggere i caratteri con un indice.
  • length() restituisce quanti caratteri contiene.
  • getline() legge una frase intera con gli spazi.
  • Non bisogna confondere char con string.
