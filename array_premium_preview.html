ARRAY

L'array è una struttura dati che permette di memorizzare più valori dello stesso tipo in una sequenza ordinata di posizioni, tutte accessibili tramite indice. È una delle basi più importanti del C++.

1. Idea chiave

Un array serve per conservare più dati dello stesso tipo usando un solo nome di variabile. Invece di creare tante variabili separate, possiamo raggrupparle in una sola struttura ordinata.

Per esempio, invece di scrivere voto1, voto2, voto3, si può creare un unico array chiamato voti e accedere ai singoli elementi con un indice.

In pratica: un array è come una fila di caselle numerate, tutte dello stesso tipo, con una dimensione decisa prima.

2. Modello mentale

Immagina un array come una fila di contenitori. Ogni contenitore ha una posizione precisa. Le posizioni sono numerate e la numerazione parte sempre da 0.

Esempio: int numeri[4] = {10, 20, 30, 40};

3. Sintassi essenziale

La forma generale di un array è questa:

tipo nomeArray[dimensione];
Dichiarazione

int numeri[5];

Crea un array di 5 interi, non ancora inizializzati in modo esplicito.

Inizializzazione

int numeri[5] = {10,20,30,40,50};

Crea l'array e assegna subito i valori.

Deduczione automatica della dimensione

int numeri[] = {10, 20, 30};

Qui il compilatore capisce da solo che la dimensione è 3.

4. Esempi guidati

Esempio 1 • Accesso agli elementi

#include <iostream>
using namespace std;

int main(){
    int numeri[4] = {10, 20, 30, 40};

    cout << numeri[0] << endl;
    cout << numeri[2] << endl;
}

L'istruzione numeri[0] legge il primo elemento. numeri[2] legge il terzo. L'indice infatti parte da 0, non da 1.

Esempio 2 • Modifica di un elemento

#include <iostream>
using namespace std;

int main(){
    int numeri[3] = {5, 8, 12};

    numeri[1] = 99;

    cout << numeri[1] << endl;
}

L'elemento all'indice 1 viene sostituito. Prima valeva 8, poi diventa 99.

Esempio 3 • Uso con un ciclo

#include <iostream>
using namespace std;

int main(){
    int voti[5] = {24, 28, 30, 27, 25};

    for(int i = 0; i < 5; i++){
        cout << voti[i] << endl;
    }
}

Questo è uno degli usi più tipici dell'array: scorrere tutti gli elementi con un ciclo for.

Esempio 4 • Caso realistico: inserimento e media

#include <iostream>
using namespace std;

int main(){
    int numeri[5];
    int somma = 0;

    for(int i = 0; i < 5; i++){
        cout << "Inserisci numero: ";
        cin >> numeri[i];
        somma += numeri[i];
    }

    cout << "Media = " << (float)somma / 5 << endl;
}

Questo esempio mostra bene quando l'array funziona bene: sappiamo già che i numeri saranno esattamente 5.

Se conosci bene indice, dimensione e ciclo for, hai già compreso il cuore del funzionamento degli array.

6. Array vs Vector

Array e vector si assomigliano, ma non vanno confusi. Entrambi usano l'indice, però l'array ha dimensione fissa mentre il vector ha dimensione variabile.

Aspetto Array Vector
Dimensione Fissa Variabile
Facilità iniziale Più semplice Più comodo ma più ricco
Libreria Non serve Serve <vector>
Quando usarlo Quando sai già quanti dati hai Quando il numero può cambiare

Quando preferire l'array

Quando il problema è semplice e la quantità di elementi è fissa e nota prima.

Quando preferire il vector

Quando l'utente può inserire un numero variabile di dati o quando vuoi più flessibilità.

7. Errori tipici degli studenti

Errore 1 • Pensare che il primo indice sia 1

In C++ il primo elemento è sempre all'indice 0, non 1.

Errore 2 • Confondere dimensione e ultimo indice

Se un array ha dimensione 5, gli indici validi sono 0,1,2,3,4. L'ultimo indice non è 5.

Errore 3 • Uscire dai limiti dell'array

int numeri[5];
numeri[5] = 100; // SBAGLIATO
Questo accesso è fuori limite, perché l'ultimo indice valido è 4.

Errore 4 • Scrivere un ciclo sbagliato

// SBAGLIATO
for(int i = 0; i <= 5; i++)

// CORRETTO
for(int i = 0; i < 5; i++)

Con <= provi a usare anche l'indice 5, che non esiste.

Errore 5 • Pensare che l'array possa crescere

No: una volta creato, l'array mantiene sempre quella dimensione.

10. Riepilogo finale

  • L'array contiene più valori dello stesso tipo.
  • Gli elementi sono ordinati e si leggono con un indice.
  • Gli indici partono sempre da 0.
  • La dimensione si decide all'inizio e poi rimane fissa.
  • L'array è molto usato con i cicli for.
  • Per quantità fisse di dati è una soluzione semplice ed efficace.
Se uno studente comprende davvero bene l'array, poi capisce con molta più facilità vector, string e molte altre strutture del C++.
